## Overview
This notebook provides a practical, hands-on tutorial on **MobileNetV1 and MobileNetV2**, two lightweight convolutional neural network architectures widely used in mobile and embedded vision applications.  
It walks through the core ideas behind depthwise separable convolutions, architectural components, and PyTorch implementation details.

## Workflow
1. Load necessary libraries and configure device (CPU/GPU)
2. Review Batch Normalization and its role in training stability
3. Understand depthwise separable convolutions and why MobileNet uses them
4. Implement MobileNetV1 building blocks in PyTorch
5. Implement MobileNetV2 inverted residual blocks
6. Assemble the full MobileNet architectures
7. Perform forward passes and analyze output shapes
8. Visualize architecture components and study computational efficiency

- Depthwise & pointwise convolutions
